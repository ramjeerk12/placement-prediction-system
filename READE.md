# 🎓 Placement Prediction System

A machine learning web application that predicts whether a student is likely to be placed based on their **CGPA** and **IQ score**.

## 🚀 Live Demo

🔗 Demo: https://your-app-name.onrender.com

## 📌 Overview

This project demonstrates an end-to-end Machine Learning workflow, from data preprocessing and model training to deployment as an interactive web application.

Users can enter their CGPA and IQ score, and the system predicts the placement outcome in real time.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/placement-predictor.git
```

Move into the project directory:

```bash
cd placement-predictor
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
